# Slack Theme — Ristretto Light Contrast

## How to apply

1. Open Slack
2. Go to **Preferences → Sidebar**
3. Scroll down to **Theme** and click **Custom theme**
4. Paste the color string below into the input field

## Color string

```
#fdf6ee,#f4e8da,#5e8e28,#fdf6ee,#f4e8da,#22211d,#29472a,#df2b0d
```

## Color mapping

| Slot | Role                        | Color     |
|------|-----------------------------|-----------|
| 1    | Sidebar background          | `#fdf6ee` |
| 2    | Sidebar item hover          | `#f4e8da` |
| 3    | Active item background      | `#5e8e28` |
| 4    | Active item text            | `#fdf6ee` |
| 5    | Sidebar hover text          | `#f4e8da` |
| 6    | Default text                | `#22211d` |
| 7    | Online/active indicator     | `#29472a` |
| 8    | Notification badge          | `#df2b0d` |
